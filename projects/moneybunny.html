Money Bunny Online Banking System

  • Tech Stack: Angular, NodeJS, Typescript, HTML, CSS, Bootstrap, Firebase
  • Website URL: Link
  • Github URL: Project Link

Web Application that allows you to manage your bank accounts and to conduct financial transactions.

Features
  • Login and Register
  • Profile Update
  • Opening a new Bank Account or adding an existing one to the App
  • Blocking or Closing an Account
  • See details about a card (IBAN, Balance, Card Type, Bank)
  • Make a simple or recurrent transaction
  • Currency Converter and Transactions Report
